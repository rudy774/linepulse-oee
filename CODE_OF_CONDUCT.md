# Code of Conduct

## Our Pledge

We want LinePulse OEE to be useful to manufacturing engineers, integrators, analysts, maintainers, students, and operators with different experience levels. We pledge to make participation respectful, practical, and welcoming.

## Expected Behavior

- Be kind and direct.
- Assume good intent while still being clear about risks and mistakes.
- Keep feedback focused on the work.
- Respect confidential plant, customer, operator, and vendor information.
- Help make examples reproducible without requiring private production data.

## Unacceptable Behavior

- Harassment, insults, or personal attacks.
- Publishing private or proprietary manufacturing data without permission.
- Sharing credentials, secrets, customer details, or operator-identifying data.
- Repeated off-topic comments that make maintenance harder.

## Enforcement

Maintainers may edit, hide, lock, or remove comments, issues, pull requests, or contributions that violate this code of conduct. Repeated or severe behavior may result in a ban from the project.

## Scope

This code applies in GitHub issues, pull requests, discussions, documentation, and other project spaces.
